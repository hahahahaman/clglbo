clglbo - common lisp + opengl = breakout

An experiment in time control (like in Braid), where state is kept tracked of
every frame and in some kind of event system that prevents state change from
happening mid-frame.

Controls:
ESC - quit

Q - rewind
W - fast-forward
E - pause
R - play

N - next level
B - previous level
M - restart level

Arrow Keys - move paddle
Space - release ball
